templategenerator by slideep

A simple passive code/template generator. Generates code once and then gives up all responsibility for it.

A passive code generator creates code that you expect to maintain by hand afterwards. Templategenerator generates code once and then gives up all responsibility for it.

Dependencies

mongocsharpdriver >= 1.3.1 FluentMongo >= 1.3.1.1 Web API for self hosting
